# imbd

## A movie database web application that utilizes the TMBD API to provide a seamless and responsive research experience. "It's Movie Night!"

### This application is powered by the TMBD API. It grants the user access to a large assortment of movies, whether old, current, or upcoming.

### There is a choice of whether to stay in or go out. Staying in allows the user to select from a wide range of movies that are available to stream on multiple platforms and save them to a favorites list for later.

### The Going Out option allows the user to view movies that are either in theaters currently or movies that are upcoming. Includes the description of the movie and places to go watch the movie based on the user's location.
